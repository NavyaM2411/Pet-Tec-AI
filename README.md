# 🐱 Pet-Tech AI 貓咪健康分析系統

## 📋 專案簡介
這是一個使用AI技術的貓咪健康分析系統，在48小時內完成開發原型，用於申請 Dr. 張中明的 TEEP 獎學金實習計劃。

### 核心功能
- 🐱 **貓咪品種識別**：辨識5種貓咪（波斯貓、暹羅貓、緬因貓、孟加拉貓、混種貓）並分析混種比例
- 🩺 **皮膚健康檢測**：分析5種皮膚狀況（健康、真菌感染、紅腫、疥癬、皮屑）
- 🛒 **智能產品推薦**：根據品種特性和皮膚狀況推薦適合的護理產品

## 🛠️ 技術架構

### 模型
- **基礎架構**：MobileNetV2（遷移學習）
- **框架**：TensorFlow / Keras
- **影像處理**：OpenCV, PIL
- **前端介面**：Gradio + 自定義CSS

### 數據集
- **總圖片數**：79張
- **品種分類**：波斯貓(5)、暹羅貓(5)、緬因貓(5)、孟加拉貓(5)、混種貓(8)
- **皮膚分類**：健康(8)、真菌(9)、紅腫(8)、疥癬(8)、皮屑(5)
- **測試圖片**：13張示範圖片
